# node-redysys-api

Node.js Redsys api implementation with new the key-hashed message authentication code (HMAC) SHA256 for the virtual payment gateway integration.

## Version
	0.0.3

## Installation

	npm install node-redsys-api --save

## Tests
	
	grunt test
